# Change Log
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/) 
and this project adheres to [Semantic Versioning](http://semver.org/).

## [Unreleased]
###Added
- Dodaj polje komponenta kompletirana
- Sejanje, usevi, vrsta useva implementirano kroz MongoDB


[0.8.0] - 2017-04-27
###Added
- Info clanci arhitektura zavrsena, sa snimanjem slika (video i fajlovi mogu slicno biti dodati).
- Snimanje novih tema, bez slika i videa.
- Froala HTML editor.
- Front-end prikaz info pages-a (samo tekstualni).
- Osnovna struktura info pages-a.
- Masine i slike masine u MongoDB.
- Osnovna struktura Node/MongoDB Back end-a postavljena.
### Changed
- Model podataka obogacen, osim za zadrugu i zadrugara

[0.7.0] - 2017-04-17
### Changed
- Kompletirano crtanje njiva jednog korisnika na Gmaps, izuzev pretvaranja u jpg i snimanja na server bazu
- Funkcionalna specifikacija obogacena.
###Fixed
- Situacija kada korinsnik ne nacrta kompletnu njivu sredjena
- Mnoge nelogicne opcije i CSS bug sredjen

[0.6.0] - 2017-04-13
### Added

- Vodic zavrsen
###Changed

##[0.2.0] - 2017-04-06
### Added
- Status notification hub
- Redux state arhitektura
- Dodavanje njive

- Prikazivanje detalja za jednu njivu
